# RAG + Ollama Demo (Node.js + TypeScript)

Простой учебный проект, который показывает базовый RAG-пайплайн:

- загрузка локальных документов из `docs/`,
- разбивка текста на чанки,
- генерация эмбеддингов через Ollama (`/api/embeddings`),
- сохранение индекса в `data/index.json`,
- простой поиск по индексу (косинусное сходство).

## Быстрый старт

```bash
npm install
# далее:
npm run build:index    # построить индекс (после написания кода)
npm run search         # выполнить поиск (после написания кода)
```
