# ТехноВолна API Документация v3.2

## Аутентификация

Все API запросы требуют ключ в заголовке:
Authorization: Bearer tv_live_sk_abc123xyz789

API ключи генерируются в личном кабинете в разделе Настройки > API Ключи.

## Эндпоинты

### POST /api/v3/quantum/encrypt

Шифрует данные квантово-устойчивыми алгоритмами.

**Параметры:**

- `data` (string, обязательный): Данные для шифрования (макс 10МБ)
- `algorithm` (string, опциональный): "qrsa-2048" или "lattice-512" (по умолчанию: "qrsa-2048")
- `callback_url` (string, опциональный): URL для webhook уведомления

**Лимит запросов:** 100 запросов в минуту
