# ТехноВолна API Документация v3.2

## Аутентификация

Все API запросы требуют ключ в заголовке:
Authorization: Bearer tv_live_sk_abc123xyz789

API ключи генерируются в личном кабинете в разделе Настройки > API Ключи.

## Эндпоинты

### POST /api/v3/quantum/encrypt

Шифрует данные квантово-устойчивыми алгоритмами.

**Параметры:**

- `data` (string, обязательный): Данные для шифрования (макс 10МБ)
- `algorithm` (string, опциональный): "qrsa-2048" или "lattice-512" (по умолчанию: "qrsa-2048")
- `callback_url` (string, опциональный): URL для webhook уведомления

**Лимит запросов:** 100 запросов в минуту

**Ответ:**

```json
{
  "encrypted_data": "qenc_a8f7d...",
  "encryption_id": "enc_1234567890",
  "processing_time_ms": 342
}
```

GET /api/v3/analytics/report/{report_id}
Получает сгенерированный аналитический отчет. Параметры:
report_id (string, обязательный): Уникальный идентификатор отчета
format (string, опциональный): "json" или "pdf" (по умолчанию: "json")
Лимит запросов: 500 запросов в минуту Коды ответа:
200: Успех
404: Отчет не найден
429: Превышен лимит запросов (повторить через 60 секунд)
Коды ошибок
E1001: Неверный API ключ
E2003: Размер данных превышает лимит
E3005: Алгоритм не поддерживается в вашем тарифе
E4002: Превышен лимит запросов
